1. var let const

1.ES5中变量的声明方式都用var，ES6中新增了let const两种变量声明方式

2.let用于声明局部作用域的变量，var用于声明全局作用域的变量(存在变量提升),const用于声明不可重新赋值的变量

3.在开发中，针对不同作用域的变量尽量用let和var分开声明，不然会有安全隐患
