2. Array 类型

数组的每一项可以保存任何类型的数据

创建数组的基本方式有两种：

注意：

1.第一种方法创建数组是，如果传递的是数值，则会按照该数值创建包含给定项数的数组；如果传递的是其他类型的参数，则会创建包含那个值的只有一项的数组

2.与对象一样，在使用数组字面量表示法时，也不会调用Array构造函数

访问数组

1.如果索引小于数组中的项数，则返回对应项的值

2.如果设置某个值的索引超过了数组现有项数，数组就会自动增加到该索引值+1的长度

3.如果将数组length属性设置为大于数组项数的值，则新增的每一项都会取得undefined值

4.数组的length属性不是只读的，可以通过设置这个属性，从数组的末尾移除项或向数组中添加新项

检测数组：

Array.isArray( value )方法用于检测数组

转换方法：

1.所有对象都具有 toLocaleString()、toString()、valueOf()方法

2.调用valueOf() 返回的还是数组本身

3.调用数组的toString()方法会返回由数组中每个值的字符串形式拼接而成的一个以逗号分隔的字符串(为了创建这个字符串会调用数组每一项的toString()方法)

4.toLocaleString()经常会返回和前两种方法相同的值，也会创建一个数组值的以逗号分隔的字符串，该方法调用的是每一项的toLocaleString()方法

join()方法

使用join()方法，可以使用不同的分隔符来构建这个字符串

join()方法只接受一个参数，即用作分隔符的字符串，如果不给join()传入任何值，或者给它传入undefined，则使用逗号作为分隔符
